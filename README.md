# location-origin.js

get location.origin

## Installation

```sh
$ npm install sasaplus1-prototype/location-origin.js
```

## Usage

via `require()`

```js
var locationOrigin = require('location-origin');
```

via `<script>`

```html
<script src="location-origin.min.js"></script>
```

### Example

```js
locationOrigin.get();
// => 'https://github.com'
```

## Functions

### get()

return value of `location.origin`.
